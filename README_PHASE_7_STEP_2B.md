# Phase 7 Step 2B — Protected Source-Inspection UI

This increment adds a developer-only Streamlit workspace backed by the
protected Step 2A source API.

Included: protected navigation, configured server-to-server API access,
safe errors, source search/status filters, aggregate metrics, document
metadata/integrity details, paginated page-aware chunks, and offline tests.

Upload, ingestion, storage download, archive/restore, and deletion remain
excluded.
